🔍 What Is It?

This tool lets you rename multiple files automatically using patterns defined in an Excel sheet.

✅ It finds and replaces text in filenames using rules
✅ Keeps a full log of what was renamed
✅ Opens the renamed folder and log file for review

🎯 Why Use It?

⚡ Fast Rename

To bulk rename files quickly

🔁 Apply Rules

To apply consistent naming using rules

📊 Use Excel

To use Excel for flexible search & replace patterns

📊 What You Need Before Running

Make sure you have:

  • A folder with files you want to rename
  • An Excel file with two columns:

Column A: old_name — the word or pattern to search

Column B: new_name — what to replace it with

📝 Example Excel:

old_name      new_name
old           new
sample        test
\d+           123

(Yes, it supports regex patterns!)

👣 How To Use It?

1

Start the Tool

Run the script (open it from your Python environment or terminal).

2

Enter Folder Path

When asked, enter the folder path where your files are.

C:\Users\YourName\Documents\MyFiles
3

Select Excel File

A popup will appear — choose your Excel file that has renaming rules.

4

Let It Work

The tool will:

  • Read the rules
  • Go through all files (including subfolders)
  • Rename matching files
  • Save a log file

📁 Output and Logs

After renaming, the tool opens:

  • The folder where files were renamed
  • A .txt log file showing the changes

Example log line:

Renamed: old_name.txt -> new_name.txt

Logs are saved here:

C:\Customization tool\rename files\

Each log file includes the date and time in its name.

🧩 Common Problems & Fixes

❌ Folder not found
Check the path and try again. Use quotes if the path has spaces.
❌ Excel file not selected
The tool needs an Excel file. Restart and select one.
⚠️ File already exists
A file with the new name already exists. Rename skipped.
🧪 Regex rule misbehaves
Make sure the pattern in old_name is written correctly.

💡 Helpful Tips

📊
