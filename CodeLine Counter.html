🔍 What Is It?

This tool helps you count meaningful lines of code (LOC) and KLOC in all .cpp and .h files inside a folder.

✅ Ignores blank lines and comments
✅ Separates count for header and source files
✅ Explains what LOC and KLOC mean in simple terms

🎯 Why Use It?

📏 Measure Size

Helps you know how much code your project contains

📊 Track Progress

Useful for tracking how much new code is written

✅ Clean Reporting

Ignores non-code lines like comments and empty lines

⚙️ How It Works

  • Reads every .cpp and .h file inside the selected folder
  • Skips lines that are:
    • Empty
    • Start with //, /*, *, */, or #
  • Counts only meaningful code lines

👣 How To Use It

1

Run the Script

Open it in your Python editor or terminal

2

Enter Folder Path

When asked, paste the full path to the folder containing your C++ files

C:\Users\You\Projects\MyCppApp
3

See Results

The tool will show LOC for each file, and total KLOC for headers and source files

📄 Understanding the Output

The tool prints two sections:

  • Header Files (LOC Count): LOC for all .h files
  • Source Files (LOC Count): LOC for all .cpp files

KLOC means "Kilo Lines of Code"

KLOC = LOC ÷ 1000

Then you will see a final summary like:

Total LOC for Header Files: 1342
KLOC = 1.34

Total LOC for Source Files: 5289
KLOC = 5.29

💡 Helpful Tips

📁

Organize Code

Keep all your source files in one folder for easy counting

💬

No Comments Counted

Lines with //, /*, or # are not counted

🚫

Empty Lines Skipped

Only actual lines of code are counted

📈

Use in Reports

Great for project size estimates or team metrics
