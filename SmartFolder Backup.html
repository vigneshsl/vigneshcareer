📁 SmartFolder Backup Tool

Easily create timestamped backups of your important folders

📌 What Is It?

This tool helps you make a safe backup of any folder with a unique timestamp. You can optionally name the backup to make it easier to identify later.

🎯 Why Use It?

  • ✅ Automatically backs up an entire folder
  • ✅ Adds date and time to the folder name to avoid overwriting
  • ✅ Lets you give the backup a custom name if you want

👣 How to Use

  1. Run the tool.
  2. When asked, paste the full path of the folder you want to back up.
  3. Choose if you want to name the backup. Type 'Y' to enter a custom name, or 'N' to skip.
  4. The tool will create a new folder under the backup location with the current date and time in its name.
  5. Once finished, the backup folder will open automatically.

📂 Where Does It Save?

Backups are saved in:

c:\Customization tool\SmartFolder Backup\

The folder will be named like this:

YourName_26_07_2025_23_44_19_123

💡 Tips for Beginners

  • ✔️ You don’t need to create folders manually — the tool does it for you
  • ✔️ You can safely run the tool multiple times — each backup has a unique name
  • ✔️ Use a short and clear name when asked (e.g., “ProjectA”) to identify backups easily

⚠️ What If Something Goes Wrong?

  • ❌ If an error occurs, make sure the folder path you gave is correct
  • 🔒 Make sure the files are not open or locked by another program

📦 Use this tool to quickly back up your work without worrying about overwriting anything.
