📦 What It Does

This batch tool performs bulk file operations (Copy, Move, or Delete) based on file extensions and source folder.

🎯 Why Use It?

  • ✅ Easily organize code files or media files
  • ✅ Save backup copies of specific file types
  • ✅ Clean up unwanted files in bulk

👣 How To Use

1

Run the Script

Double-click the CopyMoveZap.bat file or run it in the command prompt.

2

Provide Input

  • Enter the source folder path
  • Enter one or more file extensions (space-separated)
  • Select an action: Copy, Move, or Delete
  • If copying or moving, enter the destination folder
3

Wait for the Operation

The tool will go through each file and perform the selected operation.

🔧 Supported Actions

  • 📁 Copy: Duplicate matching files to another folder
  • 🚚 Move: Relocate matching files
  • 🗑️ Delete: Permanently remove matching files
⚠️ Note: Delete action is permanent. Use cautiously!

📌 Example Inputs

  • Source folder: C:\Projects\Codebase
  • File extensions: cpp h txt
  • Action: 1 (Copy)
  • Destination folder: D:\Backups\Code

This will copy all .cpp, .h, and .txt files from C:\Projects\Codebase (including subfolders) to D:\Backups\Code.

💡 Helpful Tips

🛡️

Test with Few Files

Try with a small folder before using it on large directories.

📦

Use Double Quotes

Wrap folder paths with spaces in quotes: "C:\My Folder"

🔁

Supports Multiple Extensions

You can enter multiple extensions separated by space (e.g., cpp h py).
