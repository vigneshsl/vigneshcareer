🔍 What Is It?

This tool extracts readable text (ASCII strings) from any file, even binary files.

✅ Works with .exe, .bin, .dat, and more
✅ Saves the results in readable and binary formats
✅ Automatically timestamps your output files

🎯 Why Use It?

🔍 Find Text

To discover hidden or embedded strings inside files

🛠 Debug Info

To extract debug messages from binary files

📄 Recover Data

To recover readable data from unknown files

📂 What You Need

  • Any file you want to scan
  • Python installed
  • This script saved and ready

No Excel or extra setup required!

👣 How To Use It

1

Start the Script

Run it from Python or double-click to open it

2

Select Your File

A file dialog will pop up — choose the file you want to scan

3

Let It Extract

The tool will extract readable strings and save two output files:

  • ✔️ A text file (.txt)
  • ✔️ A binary file (.bin)

📁 Output Example

If you scan a file like firmware.bin, output may look like:

C:\Customization tool\dump\firmware_20250726_230103.txt  
C:\Customization tool\dump\firmware_20250726_230103.bin
📝 Note: Files are saved with the current date and time in the name.

🧩 Common Problems & Fixes

❌No file selected
Run again and select a file when the dialog appears
❌File not found
Make sure the file still exists before selecting
❌Write error
Check if the destination folder is locked or full
