🔎 What Is It?

This tool helps you automatically search and replace content inside files using rules defined in an Excel or CSV file.

🎯 Why Use It?

  • ✅ Replace multiple patterns across many files
  • ✅ Use Excel or CSV for flexible rule setup
  • ✅ Supports skipping lines by setting replacement as blank

✨ Features

  • ✅ Replace text in `.cpp`, `.h`, `.txt`, or any text file
  • 📄 Uses a simple Excel or CSV file as input
  • 🚫 Skips lines if replacement value is empty
  • 🔍 Applies replacements safely with previews in console

👣 How To Use It?

1

Run the Tool

Open the Python file using your IDE or terminal.

2

Select the Excel or CSV File

Use the popup window to select the file containing your replacement rules.

3

Select Target Files

Choose the files where replacements should be made (can be `.cpp`, `.h`, `.txt`, etc.).

4

Replacement Done!

After processing, the tool shows a confirmation and prints replaced lines in the console.

📝 Replacement Rules Format

The Excel or CSV file must have two columns:

old_content | new_content

Examples:

old_content,new_content
temp,temperature
foo,bar
remove_line,

Note: If new_content is empty, that entire line will be removed from the file.

🧩 Common Problems & Fixes

❌ Excel or CSV not selected
You must choose a mapping file before continuing.
❌ Invalid columns
Check that your Excel/CSV has old_content and new_content columns.
⚠️ File encoding error
Make sure files are UTF-8 encoded or use a plain text format.

💡 Helpful Tips

🧪

Test First

Try on 1-2 files first to verify your rules.

📝

Use Clean Data

Trim extra spaces in Excel cells to avoid mismatches.

💾

Backups

Keep a copy of original files in case you want to undo changes.

📂

Batch Files

You can select multiple files at once for faster processing.
