🧹 Comment Remover Tool

Clean up your code by removing // comments from text files

📌 What Is It?

This tool helps you clean up source or log files by removing all lines that start with // (single-line comments), and also removes inline comments from code lines.

🎯 Why Use It?

  • ✅ Quickly remove unnecessary comments from files
  • ✅ Make your files cleaner and easier to review
  • ✅ Saves the cleaned file separately — your original file stays safe

👣 How To Use

  1. Run the tool.
  2. When prompted, select the file you want to clean.
  3. The tool will automatically:
    • Skip lines that start with //
    • Remove // comments from the middle of lines
    • Keep only meaningful, non-empty lines
  4. A new file will be created with "_cleaned" added to the original name.
  5. The cleaned file will be highlighted in your file explorer automatically.

📂 Output Location

The cleaned file is saved in the same folder as the original file, with _cleaned added to the filename.

Example: original.txt → original_cleaned.txt

💡 Tips for Beginners

  • ✔️ Use this tool before sharing code or logs — it makes them cleaner.
  • ✔️ Don’t worry, your original file won’t be changed.
  • ✔️ Works with .txt and many types of source files too!

⚠️ Common Issues

  • ❌ If you see “No file selected,” try running the tool again and choose a valid file.
  • ❌ Make sure the file is not open in another program while cleaning.
