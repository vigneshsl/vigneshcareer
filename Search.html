🔎 What Is It?

This tool helps you search for multiple terms in all files inside a folder.

  • ✅ Search for file names
  • ✅ Search by file extension
  • ✅ Search inside the file content

🎯 Why Use It?

🧠 Search Smarter

Find files and content based on multiple terms

📂 Save Time

Quickly search through folders and subfolders

📋 Organized Output

Automatically saves a result file for easy access

✨ What Can It Do?

  • ✅ Search by filename
  • ✅ Search by extension (like .txt, .cpp)
  • ✅ Search inside the file content
  • 📁 Shows full file path for matches
  • 📊 Saves results in a CSV format

👣 How To Use?

1

Run the Tool

Open the script using Python or double-click the `.py` file if it's executable.

2

Enter Search Terms

When the tool asks for input, type your search words separated by commas.

📥 What Can You Type?
You can enter:
  • 📄 File names (e.g., report, main)
  • 🧩 Extensions (e.g., .cpp, .txt)
  • 🔍 Words inside files (e.g., error, port=)

Example Input:

error, .cpp, config

This will search for:

  • Files with “error” or “config” in their name
  • Files that end in .cpp
  • Any file that contains the word “error” or “config” inside
3

Select Folder

A folder dialog will appear – just select the folder you want to search through.

4

View the Results

After the search, results will be saved and the folder will open automatically.

You’ll get a file like:

multi_search_results_20250726.csv

📁 Where Are the Results Saved?

The tool automatically creates a folder like:

C:\Customization tool\search Exetion\

Inside that folder, you will find the `.csv` file with all the results.

🧩 Common Problems & Fixes

❌ No input provided
Make sure to type at least one search term.
❌ Folder not selected
You must choose a folder to search.
⚠️ No matches found
Check your search terms – they might not exist in files.
